# CHRONO — Premium Stopwatch

CHRONO is a static stopwatch app with lap tracking built using HTML, CSS, and JavaScript.

## Deployment

- Open `index.html` in any modern browser.
- GitHub Pages deployment is included via `.github/workflows/pages.yml`.

## Included Files

- `index.html` — app markup
- `style.css` — styling and responsive layout
- `script.js` — stopwatch logic, lap tracking, and UI interactivity
- `package.json` — project metadata (version `1.0.0`)
- `.gitignore` — ignored files for repository cleanliness
- `.github/workflows/pages.yml` — GitHub Pages deployment workflow

## Version

This project is version `1.0.0`.
